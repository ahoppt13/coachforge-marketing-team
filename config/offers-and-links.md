## X (Twitter) link rule
- **Every X post carries a link** (X allows links and the DM-keyword funnel is weak there).
- If the post promotes the **free phone walkthrough** (phone-setup / "Comment PHONE" angle), use the **walkthrough link**.
- Otherwise use the **Gumroad profile link**.
- The link replaces the "Comment KEYWORD" CTA on X (keep keyword CTAs on IG/TikTok/FB where the DM funnel works).
- Keep the whole post **≤ 280 characters including the link**. The walkthrough URL is ~52 chars raw — budget for it; if it won't fit, tighten the body, never the link.

## Cadence
- **X posts daily** — one X post every day of the week, in addition to the carousel/video posts on IG/TikTok. If a day's main post is IG/TikTok-only, still write a standalone X post for that day (repurpose the same angle into ≤280 with a link).

## Other networks
- Instagram / TikTok / Facebook: keep the **Comment KEYWORD → DM** CTA (that funnel works there); do **not** swap in a raw link unless Aaron asks.
- Keywords in use: `PHONE` (phone walkthrough), `ADMIN` (admin starter prompts), `PROMPTS` (prompt pack), `LEVERAGE` (30-day roadmap).
